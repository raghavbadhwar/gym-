# Credity Evidence Pack

This directory stores release proof artifacts.

## Structure
- `01-access-control/`
- `02-ci-cd/`
- `03-security/`
- `04-observability/`
- `05-backup-restore/`
- `06-incidents/`
- `07-slo/`

## Rules
- Evidence must map to current release SHA.
- Include timestamp + owner for each artifact.
- No secrets in evidence files.
