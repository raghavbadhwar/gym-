# CredVerse Ecosystem README

CredVerse is a monorepo containing Issuer, Recruiter, Wallet, Gateway, and shared authentication packages for digital credential issuance and verification.

## Applications

- `CredVerseIssuer` - issues verifiable credentials for institutions.
- `CredVerseRecruiter` - verifies candidate credentials for hiring workflows.
- `BlockWalletDigi` - stores and shares user credentials.
- `credverse-gateway` - public-facing ecosystem entry point.
- `packages/shared-auth` - shared authentication and security utilities.
